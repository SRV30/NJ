## 💎 Overview

**Nandani Jewellers** is a full-stack, production-grade web platform developed as a commissioned freelance project for a premium jewellery brand.

The application combines **luxury UI design with robust backend architecture**, delivering a seamless experience for users and powerful tools for business operations.

> *A real-world product built for real users — not just a demo project.*

---

## 🎯 Project Highlights

- 🏆 **End-to-End Ownership** — Designed, developed, and deployed independently  
- 🌍 **Live Production App** — Actively used by real customers  
- 🔐 **Secure Authentication** — JWT + OTP email verification  
- 🔔 **Real-Time Admin Alerts** — Instant notifications on new users  
- 📱 **Fully Responsive** — Optimized across all devices  
- 🎨 **Luxury UI/UX** — Designed to reflect premium jewellery branding
